# Toybox

A collection of fun browser games for kids. Designed with my 4-year-old and Claude.

## Games

- **A-maze-ing!** - Navigate a mouse through randomly generated mazes to find the cheese
- **Feather Collect** - Move leaves to discover hidden feathers
- **Powder Play** - Draw with sand, water, and fire in a physics sandbox

## Running

Open `index.html` in any modern browser. No installation or build step required.

## Adding Games

Create a new folder under `games/` with `index.html`, `style.css`, and `game.js`, then add a link card to the root `index.html`.
